A simple database program. The main focus of the assignment was object-oriented development ad unit testing, hence why user commands are limited.

Usage:
java Db





The available commands are:



help - display commands



add table - Adds a new Table to the Database. The user is prompted for a table name and column names(which must be unique)



add record <tablename> - Adds a new Record to the specified table. The user is prompted for field values - the number of fields must match the number of columns in the table and all values are entered as Strings



print <tablename> - Prints the specified table

print all - Prints all tables in the database


exit - Exits the program
